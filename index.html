El juego que seleccionamos como grupo es "Minecraft" el competidor directo de este juego es Terraria, un juego que se lanzo al mismo tiempo que Minecraft y supo ser por muchos años el competidor.

Minecraft es un videojuego de mundo abierto y supervivencia donde los jugadores exploran un escenario generado de manera infinita, recolectan recursos, construyen estructuras, fabrican herramientas y se enfrentan a criaturas hostiles. No tiene un objetivo fijo: se puede jugar de forma creativa, centrado en construir y explorar, o en modo supervivencia, gestionando salud y hambre mientras se avanza hacia desafíos como derrotar a un dragón.

Requisitos minimos de este juego

CPU: Intel Core i3-3210 o AMD A8-7600 (o equivalente) • RAM: 1 GB para gráficos básicos (pero para rendimiento aceptable se recomiendan al menos 4 GB) • GPU integrada: Intel HD Graphics 4000 (Ivy Bridge) o AMD Radeon R5 series, compatible con OpenGL 4.4 • GPU discreta (alternativa mínima): GeForce serie 400 o Radeon HD 7000 series, también con OpenGL 4.4 • Almacenamiento: aprox. 1 GB libre para la instalación básica; los mapas, mods, etc. requieren más espacio. • Sistema operativo: Windows 10 de 64 bits, macOS Mojave (10.14.5) o posterior; Linux 64-bits moderno.

Requisitos recomendados para este juego

• CPU: Intel Core i5-4690 o AMD A10-7800 (o equivalente) • RAM: 8 GB • GPU: tarjeta discreta moderna con soporte para OpenGL 4.5; por ejemplo la GeForce serie 700 o la Radeon R7 / Rx 200 series. • Almacenamiento: ~4 GB libres, preferiblemente SSD para mejor carga y tiempos de respuesta.
