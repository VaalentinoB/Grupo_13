El juego que seleccionamos como grupo es "Minecraft" el competidor directo de este juego es Terraria, un juego que se lanzo al mismo tiempo que Minecraft y supo ser por muchos años el competidor.

Minecraft es un videojuego de mundo abierto y supervivencia donde los jugadores exploran un escenario generado de manera infinita, recolectan recursos, construyen estructuras, fabrican herramientas y se enfrentan a criaturas hostiles. No tiene un objetivo fijo: se puede jugar de forma creativa, centrado en construir y explorar, o en modo supervivencia, gestionando salud y hambre mientras se avanza hacia desafíos como derrotar a un dragón.

Requisitos mínimos de este juego

  • CPU: Intel Core i3-3210 o AMD A8-7600 (o equivalente)
  • RAM: 1 GB para gráficos básicos (recomendado 4 GB)
  • GPU integrada: Intel HD Graphics 4000 o AMD Radeon R5 series
  • GPU discreta (opcional): GeForce serie 400 o Radeon HD 7000 series
  • Almacenamiento: 1 GB libre (más para mapas y mods)
  • Sistema operativo: Windows 10 64-bits, macOS Mojave o Linux 64-bits

    • Requisitos minimos para stremear este juego

  • CPU: Intel Core i5 (8ª o 9ª generación) o AMD Ryzen 5 serie 3000
  • Memoria RAM: 16 GB
  • Tarjeta gráfica: NVIDIA GTX 1650 o AMD equivalente (compatible con OpenGL 4.5)
  • Almacenamiento: SSD con al menos 4 GB libres
  • Conexión a internet: mínimo 5-10 Mbps de subida

Requisitos recomendados para este juego

  • CPU: Intel Core i5-4690 o AMD A10-7800 (o equivalente)
  • RAM: 8 GB
  • GPU: tarjeta discreta moderna con soporte para OpenGL 4.5 (por ejemplo, NVIDIA GTX 750 o superior / AMD equivalente)
  • Almacenamiento: ~4 GB libres, preferiblemente SSD

    • Requisitos recomendados para stremear este juego

  • CPU: Intel Core i7 de última generación o AMD Ryzen 7
  • Memoria RAM: 24 GB o 32 GB
  • Tarjeta gráfica: NVIDIA GTX 1660 Super, RTX 2060 / 3060 o AMD equivalente
  • Almacenamiento: SSD NVMe rápido con espacio libre para grabaciones
  • Conexión a internet: 15 Mbps de subida o superior
  • Extras: uso de codificación por hardware (NVENC en NVIDIA o AMF en AMD) para optimizar el streaming
